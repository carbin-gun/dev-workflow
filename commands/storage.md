name: storage
description: Data model and storage design for backend systems
---

Use the `dev-workflow:storage` skill to design data models and storage.

This skill will:
- Identify entities and their relationships
- Define fields, constraints, and indexes
- Choose appropriate storage technologies
- Generate `docs/storage/<system>.md` per backend system

Prerequisites: `docs/requirements.md` and `docs/architecture.md` must exist.

Invoke the dev-workflow:storage skill and follow it exactly as presented to you
